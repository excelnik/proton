# פרוטון — מערכת ניהול פיננסי אישי 💰

> גרסת בטא פעילה | הפרויקט בפיתוח

אפליקציית דסקטופ לניהול פיננסי אישי, בנויה עם Electron + React + SQLite. כל הנתונים נשמרים מקומית על המחשב — ללא ענן, ללא שרתים חיצוניים.

---

## התקנה והרצה

### דרישות מקדימות
- [Node.js](https://nodejs.org) גרסה 18 ומעלה

### שלבי התקנה

```bash
git clone https://github.com/excelnik/proton.git
cd proton
npm install
npx electron-rebuild -f -w better-sqlite3
npm run dev
```

## פיצ'רים קיימים

### ניהול בסיסי
- **דשבורד** — KPI, בקרת תקציב, שווי נקי, גרפים, מעשרות ומוטו
- **חשבונות** — ניהול חשבונות בנק, אשראי ומזומן עם יתרות מחושבות
- **תנועות** — צפייה, הוספה, עריכה inline (מהירה ומלאה), אקורדיון שדות מקושרים

### ייבוא נתונים
- **ייבוא מהבנק** — ייבוא קבצי Excel/CSV עם מיפוי עמודות, זיהוי כפילויות וסיווג אוטומטי
- **הדבקת טקסט** — ייבוא ישיר מהעתקה מאקסל

### ניתוח וניהול
- **סיווג קטגוריות** — סיווג תנועות עם למידה אוטומטית
- **קטגוריות** — ניהול היררכי עם תת-קטגוריות, אייקונים וצבעים
- **תכנון תקציב** — Zero-based budgeting עם auto-save וממוצע 3 חודשים
- **מעשרות** — חישוב מעשר/חומש עם ניכוי הוצאות פטורות

### פעולות מתקדמות על תנועות
- **שכפול** — יצירת תנועה זהה במהירות
- **פיצול** — חלוקת תנועה למספר קטגוריות
- **קיזוז** — חיבור תנועות מקזזות עם מנוע שאריות

### התחייבויות
- **הלוואות** — לוח סילוקין שפיצר Runtime, תנועות וירטואליות, מעקב תשלומים
- **ביטוחים** — מעקב פוליסות, חידוש עם השוואת מחירים
- **תשלומים והו"ק** — הוראות קבע ועסקאות בתשלומים עם תנועות וירטואליות

### חיסכון ושווי
- **יעדי חיסכון** — מעקב התקדמות, הפקדות, חישוב PMT
- **שווי נקי** — נכסים, התחייבויות, קרן חירום, חובות זמניים

### הגדרות
- **חוקי זיהוי** — ניהול חוקי סיווג אוטומטי
- **גיבוי ושחזור** — ייצוא וייבוא בסיס הנתונים
- **הגדרות מערכת** — תאריכים, מעשרות, קרן חירום

## ארכיטקטורה

```
proton/
├── electron/         # Electron main process
├── src/
│   ├── db/           # SQLite schema & queries
│   ├── components/   # Sidebar
│   └── pages/        # מסכי האפליקציה
├── assets/           # אייקונים ומשאבים
└── public/           # HTML entry point
```

**סטאק:** Electron · React · better-sqlite3 · Recharts · Webpack · Babel
